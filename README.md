# expanded-tables
Expanded Tables Chrome Extension

This is a simple Chrome extension to aid in the structural assessment of HTML table elements.

# Testing:
Go to the extensions page in Chrome by typing chrome://extensions/ in the omnibox, or clicking the menu > More tools > Extensions. 
Click the "Developer mode" box in the upper right-hand corner of the page. Click "Load unpacked extension..." and choose the directory containing your extension files. This will load your extension in Chrome for testing.
